**トラッカーの追加**

トラッカーはチケットの種類（バグ・機能・タスクなど）を定義します。

1. **「管理」** → **「トラッカー」** → **「新しいトラッカー」**
2. 名称・デフォルトステータス・説明を入力
3. 表示するフィールド（期日・予定工数・親チケットなど）を選択
4. 保存後、各プロジェクトの設定でトラッカーを有効化する

**カスタムフィールドの追加**

標準フィールド以外に独自の入力項目を追加できます。

1. **「管理」** → **「カスタムフィールド」** → **「新しいカスタムフィールド」**
2. 対象（チケット・プロジェクト・ユーザーなど）を選択
3. 形式（テキスト・リスト・日付・チェックボックスなど）を選択
4. 適用するトラッカーとプロジェクトを指定して保存

**カスタムフィールドの例**

| 用途 | 形式 |
|------|------|
| 顧客名 | テキスト |
| 対応ブラウザ | リスト（複数選択） |
| 承認フラグ | チェックボックス |
| 顧客確認日 | 日付 |

> カスタムフィールドを追加しすぎるとチケット画面が煩雑になります。本当に必要な項目に絞ることを推奨します。
