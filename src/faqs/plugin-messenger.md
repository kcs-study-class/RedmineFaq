---
title: "Messenger プラグインの使い方（Slack・Discord などへの通知連携）"

---

Redmine のイベント（チケット作成・更新など）を Slack・Discord・Rocket.Chat・Mattermost に通知するプラグインです。

**対応サービス**

- Slack
- Discord
- Rocket.Chat
- Mattermost

**Webhook URL の取得**

**Slack の場合：**
1. Slack の App 管理ページ → **「Incoming Webhooks」** を作成
2. 通知先チャンネルを選択して Webhook URL をコピー

**Discord の場合：**
1. サーバー設定 → **「連携サービス」** → **「ウェブフック」**
2. 新しいウェブフックを作成して URL をコピー

**Redmine 側の設定**

1. 管理画面 → **「プラグイン」** → Messenger の **「設定」** を開く
2. 使用するサービスを選択し Webhook URL を入力
3. 通知するイベントを選択する

| イベント | 説明 |
|----------|------|
| チケット作成 | 新規チケットが作成されたとき |
| チケット更新 | ステータス・担当者などが変更されたとき |
| コメント追加 | チケットにコメントが投稿されたとき |
| Wiki 更新 | Wiki ページが編集されたとき |

4. **「保存」**

**プロジェクトごとの設定**

プロジェクトの **「設定」** → **「Messenger」** タブから、プロジェクト単位で通知先チャンネルを変更できます。

> 全イベントを通知するとノイズが増えるため、最初は「チケット作成」と「ステータス変更」のみに絞ることを推奨します。
