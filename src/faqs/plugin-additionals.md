---

Redmine のカスタマイズ性を高める多機能プラグインです。ダッシュボード、Wiki マクロ、各種 UI 改善を提供します。

**ダッシュボードの設定**

1. 右上のユーザーアイコン → **「マイページ」** を開く
2. **「ダッシュボードを編集」** をクリック
3. ウィジェットを追加・並び替えしてレイアウトをカスタマイズ
4. **「保存」** で確定

**Wiki マクロの使い方**

Wiki ページや説明欄で以下のマクロが使用可能になります。

| マクロ | 説明 |
|--------|------|
| `{{hello_world}}` | 動作確認用サンプルマクロ |
| `{{issue(#123)}}` | 指定チケットへのリンクを自動生成 |
| `{{user(login)}}` | ユーザー名を表示 |
| `{{last_updated_at}}` | ページの最終更新日時を表示 |

**グローバル設定**

管理画面 → **「プラグイン」** → Additionals の **「設定」** から以下を変更できます。

- テキストフォーマットのカスタマイズ
- サイドバーの非表示設定
- 追加フィールドの表示制御

> 他の AlphaNodes プラグイン（Messenger・Privacy & Terms・SAML）との連携機能も含まれています。
