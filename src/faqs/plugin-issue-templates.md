トラッカーごとにチケットのひな形（テンプレート）を作成し、新規チケット作成時に呼び出せるプラグインです。

**テンプレートの作成**

1. プロジェクトの **「設定」** → **「チケットテンプレート」** タブを開く
2. **「新しいテンプレート」** をクリック
3. 以下の項目を設定する

| 項目 | 説明 |
|------|------|
| テンプレート名 | 選択時に表示される名前 |
| トラッカー | このテンプレートが適用されるトラッカー |
| 題名 | チケット題名の初期値（変数使用可） |
| 説明 | チケット説明の初期値（Markdown 可） |
| 担当者・優先度 | 各フィールドのデフォルト値 |

4. **「保存」**

**テンプレートの使い方**

1. **「新しいチケット」** をクリックする
2. トラッカーを選択すると **「テンプレートを選択」** ドロップダウンが表示される
3. 使用したいテンプレートを選ぶと各フィールドに初期値が入力される
4. 必要に応じて修正して **「作成」**

**グローバルテンプレート（管理者）**

管理画面 → **「チケットテンプレート」** から全プロジェクト共通のテンプレートを作成できます。

**活用例**

- バグ報告テンプレート：「発生環境」「再現手順」「期待結果」「実際の結果」を説明欄に定型文として挿入
- 機能要望テンプレート：「目的」「要件」「受け入れ条件」を挿入

> テンプレートを使うと入力漏れが減り、チケットの品質が均一化されます。
