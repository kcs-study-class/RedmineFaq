Redmine にプライバシーポリシーの同意バナーや利用規約ページを追加できるプラグインです。

**Cookie 同意バナーの設定**

1. 管理画面 → **「プラグイン」** → Privacy & Terms の **「設定」** を開く
2. 以下を設定する

| 項目 | 説明 |
|------|------|
| Cookie バナー | 有効 / 無効 |
| バナーテキスト | 表示するメッセージ |
| プライバシーポリシー URL | 外部リンクまたは内部ページの URL |
| 同意ボタンのラベル | 「同意する」など |

3. **「保存」** でサイト下部にバナーが表示される

**利用規約ページの作成**

1. 設定画面で **「利用規約」** タブを開く
2. Markdown または HTML で規約本文を入力
3. ユーザー登録時に規約への同意を必須にするかどうかを設定

**ユーザー登録時の同意取得**

利用規約を有効にすると、セルフ登録画面に「利用規約に同意する」チェックボックスが追加されます。

**プライバシーポリシーリンクの表示**

フッターにプライバシーポリシーへのリンクを自動で追加します。

> GDPR や個人情報保護法への対応が必要な場合に活用できます。法的な文書内容は法務担当者に確認してください。
