メンテナンス情報や重要なお知らせをサイト全体に表示できるプラグインです。

**バナーの設定（管理者のみ）**

1. 管理画面 → **「バナー設定」** を開く
2. 以下の項目を設定する

| 項目 | 説明 |
|------|------|
| バナー表示 | 有効 / 無効 |
| バナーテキスト | 表示するメッセージ（HTML 可） |
| 表示対象 | 全ユーザー / ログイン済みユーザーのみ |
| バナーの色 | 背景色・文字色 |

3. **「保存」** をクリックすると即時反映される

**活用例**

- `【メンテナンス】○月○日 22:00〜24:00 はサービスを停止します`
- `新機能リリースのお知らせ：チケットテンプレート機能を追加しました`
- `【重要】パスワードポリシーが変更されました。次回ログイン時に変更をお願いします`

**バナーの解除**

管理画面 → **「バナー設定」** → バナー表示を **「無効」** に変更して保存します。

> バナーはすべてのページの上部に表示されます。重要度の低いお知らせは頻繁に使いすぎないようにしましょう。
