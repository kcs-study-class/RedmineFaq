---

Redmine 標準のガントチャートを大幅に強化するプラグインです。ドラッグ操作や依存関係の可視化が可能になります。

**ガントチャートを開く**

プロジェクトメニューの **「Easy Gantt」** をクリックします。

**基本操作**

| 操作 | 方法 |
|------|------|
| バーを移動 | チケットバーをドラッグ＆ドロップ |
| 期間を変更 | バーの左右端をドラッグ |
| チケットを追加 | ガント上で空白行をクリック |
| 依存関係の設定 | バーの端から別のバーへ矢印をドラッグ |

**表示フィルター**

- 担当者・優先度・ステータスで絞り込み可能
- 折りたたみ表示でサブチケットを管理

**PRO 版との違い**

無料版（Light）と PRO 版の主な違いは以下のとおりです。

| 機能 | Light | PRO |
|------|-------|-----|
| ドラッグ移動 | ○ | ○ |
| 依存関係 | △（表示のみ） | ○（設定可） |
| クロスプロジェクト表示 | ✕ | ○ |
| リソース管理 | ✕ | ○ |
| エクスポート（PDF/PNG） | ✕ | ○ |

**設定**

管理画面 → **「プラグイン」** → Easy Gantt の **「設定」** から、デフォルト表示期間やカラーテーマを変更できます。

> チケットに「開始日」と「期日」が設定されていないとガント上に表示されません。
