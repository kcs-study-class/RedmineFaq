---
title: "新しいプロジェクトを作成するには？"
category: "プロジェクト管理"
order: 1
---

プロジェクトの作成には **管理者** または **プロジェクト作成権限** が必要です。

1. 上部メニューの **「プロジェクト」** → **「新しいプロジェクト」** をクリック
2. 以下の項目を入力する

| 項目 | 説明 |
|------|------|
| 名称 | プロジェクトの表示名 |
| 識別子 | URL に使われる英数字の ID（作成後変更不可） |
| 説明 | プロジェクトの概要（任意） |
| 公開 | チェックするとログインなしで閲覧可能 |

3. 使用するモジュール（チケット管理・ガント・ウィキなど）にチェックを入れる
4. **「作成」** をクリック

作成後、メンバーの追加やトラッカー・カテゴリの設定を行ってください。
