---
title: "Redmine Message Customize プラグインの使い方（表示メッセージのカスタマイズ）"
category: "プラグイン"
order: 11
---

Redmine 上に表示される各種メッセージ（ラベル・説明文など）を管理画面から書き換えられるプラグインです。

**メッセージのカスタマイズ**

1. 管理画面 → **「メッセージカスタマイズ」** を開く
2. 変更したいメッセージのキーを検索する
3. 上書きしたい文字列を入力して **「保存」**

**活用例**

| 変更前（デフォルト） | 変更後（カスタム例） |
|---------------------|---------------------|
| 「チケット」 | 「タスク」 |
| 「ウォッチャー」 | 「関係者」 |
| 「担当者」 | 「アサイン先」 |
| 「期日」 | 「完了予定日」 |

**言語ごとの設定**

ログイン言語（日本語・英語など）ごとに個別にメッセージを設定できます。

**設定のリセット**

カスタマイズしたメッセージを元に戻すには、該当のカスタム値を空にして保存します。

> 用語を社内の慣習に合わせることで、Redmine をより直感的に使いやすくできます。
