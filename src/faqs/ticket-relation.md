**親チケット・子チケット**

大きなタスクをサブタスクに分割する場合に使います。

- チケット編集画面の **「親チケット」** フィールドにチケット番号を入力する
- 親チケットの進捗率は子チケットの平均値で自動計算される

**チケット間の関連付け**

チケット詳細画面の **「関連するチケット」** セクションで以下の関係を設定できます。

| 関係の種類 | 説明 |
|-----------|------|
| 関連している | 単純な関連（参照） |
| 複製された | 別チケットと重複している |
| ブロックしている | このチケットが完了しないと先に進めない |
| ブロックされている | 別のチケットが完了するまで着手できない |
| 先行する | このチケットが先に完了する予定 |
| 後続する | 別チケットの後に開始する予定 |

「ブロック」や「先行・後続」を設定すると、ガントチャートに依存関係が反映されます。
