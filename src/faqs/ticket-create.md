---
title: "チケットの作成方法を教えてください"
category: "チケット管理"
order: 1
---

1. プロジェクトを開き、上部の **「チケット」** タブをクリック
2. 右上の **「新しいチケット」** をクリック
3. 以下の項目を入力する

| 項目 | 説明 |
|------|------|
| トラッカー | チケットの種類（バグ・機能・サポートなど） |
| 題名 | チケットのタイトル（必須） |
| 説明 | 詳細情報。Markdown 記法が使用可 |
| ステータス | 初期値は「新規」 |
| 優先度 | 低・通常・高・急いで・今すぐ |
| 担当者 | プロジェクトメンバーから選択 |
| 期日 | 完了予定日 |
| 予定工数 | 見積もり時間（h） |
| 親チケット | サブチケットにする場合に指定 |

4. **「作成」** をクリックして保存

> ファイルの添付は説明欄下部の「ファイル」エリアにドラッグ＆ドロップで行えます。
